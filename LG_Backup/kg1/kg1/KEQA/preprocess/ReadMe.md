### Folder Description
- This folder stores files generated in preprocessing steps. 
- Models generated at subsequent steps are also stored in this folder.
##
- Either run programs 1)train_detection.py ,2)train_entity.py 3)train_pred.py or download pre-trained models can be downloaded from [link](https://drive.google.com/file/d/10P7EFDk_UCGUu9ENGMKOF5UyCVrlU5Ei/view?usp=sharing)
- train_detection.py saves the model in dete_best_model.py
- train_entity.py saves the model in entity_best_model.py
- train_pred.py saves the model in pred_best_model.pt
